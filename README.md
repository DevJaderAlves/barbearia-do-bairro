# 💈 Barbearia do Bairro

Projeto desenvolvido com foco em apresentar uma barbearia moderna, com interface elegante e navegação intuitiva.

---

## 🚀 Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  
- React  
- Node.js  

---

## ⚙️ Funcionalidades

- Apresentação da barbearia
- Sessão de equipe
- Galeria de cortes
- Layout responsivo
- Botão de contato via WhatsApp
